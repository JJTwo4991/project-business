## 실행 방법

### Step 1. 사이트 접속
1. Chrome 브라우저로 접속: https://franchise.ftc.go.kr/mnu/00013/program/userRqst/list.do
2. 아무 업종이나 선택 후 **검색** 클릭 (페이지가 로드되어야 함)
3. `F12` → **Console** 탭 열기

### Step 2. 스크립트 실행
- `ftc_scrape_all.js` 파일 내용을 **전체 복사** → Console에 **붙여넣기** → Enter
- 자동으로 13개 업종을 순차 스크래핑합니다

### Step 3. 기다리기
- **예상 소요 시간**: 브랜드당 ~15초 × 총 ~3,500+개 = **약 15~18시간**
- 컴퓨터 **절전 모드 끄기** 필수! (절전되면 중단됨)
- 20개마다 중간 CSV 자동 다운로드
- 업종 완료 시 해당 업종 CSV + JSON 자동 다운로드

## 중단 후 이어서 하기

### 방법 1: 같은 브라우저 탭에서
콘솔 출력에서 마지막으로 성공한 위치를 확인한 뒤:
```javascript
window._resumeIndustry = 2;  // 3번째 업종부터 (0부터 시작)
window._resumeBrand = 50;    // 그 업종의 51번째 브랜드부터 (0부터 시작)
// 스크립트 다시 붙여넣기
```

### 방법 2: 브라우저를 껐다 다시 열었을 때
1. 사이트 다시 접속 + 검색
2. 콘솔에서:
```javascript
window._resumeIndustry = 2;
window._resumeBrand = 50;
// 스크립트 붙여넣기
```

### 방법 3: 다른 컴퓨터에서 이어서 하기
1. 이전 컴퓨터에서 다운로드된 JSON 파일들 복사
2. 새 컴퓨터에서 사이트 접속 + 검색
3. 콘솔에서:
```javascript
// 이전 결과 로드 (JSON 파일 내용을 붙여넣기)
window._ftcAllResults = {
    "pizza": [/* ftc_pizza_costs_final.json 내용 */],
    "chicken": [/* ftc_chicken_costs_final.json 내용 */]
};
window._resumeIndustry = 2;  // 다음 업종 인덱스
window._resumeBrand = 0;     // 처음부터
// 스크립트 붙여넣기
```

|------|------|
| `ftc_{업종}_costs_final.csv` | 업종별 최종 CSV (탭 구분) |

## 주의사항
- **Rate limit**: 브랜드 간 12~18초 랜덤 딜레이, 업종 간 120초 쿨다운
- **차단 감지**: 응답 50KB 미만이면 차단으로 판단 → 90초 대기 후 재시도
- **연속 차단 3회**: 자동 중단 + 결과 저장 → 10분 후 이어서 실행
- **절전 모드**: 반드시 비활성화. 절전 시 스크립트 중단됨
- **view.do 응답**: 정상 3MB+, 50KB 미만이면 차단된 것
