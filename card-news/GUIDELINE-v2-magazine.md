# 카드뉴스 가이드라인 v2 — 매거진 에디토리얼 스타일

> 이 문서만 보고 14개 업종의 매거진 스타일 카드뉴스를 무제한 생산할 수 있도록 작성됨.

---

## 1. 형식

- **비율**: 1080×1920 (9:16 릴스/캐러셀)
- **카드 수**: 10장 (인스타 캐러셀 최대)
- **톤**: 공포마케팅 + 어그로 후킹, 거대 숫자 중심
- **템플릿**: `templates/card-template-v2-magazine.html`

---

## 4. 계산 공식

### 고정 상수

| 상수 | 값 | 근거 |
|------|-----|------|
| 영업일 | **26일** | `DEFAULT_OPERATING_DAYS` |
| 사장 근무시간 | **12시간/일** | 자영업자 평균 추정 |
| 최저시급 (2026) | **10,450원** | 고용노동부 |
| 인건비 (직원 1명) | **240만원** | 주 40시간 × 최저시급 + 주휴수당 (4대보험 미포함) |
| 임대료 | **150만원** | 중형 서울 추정 |
| 대출이자 | **44만원** | 1.5억 × 연 3.5% ÷ 12 |

### 변동 공식

```
월매출        = avg_daily_customers_medium × avg_ticket_price × 26
재료비        = 월매출 × material_cost_ratio
배달수수료     = 월매출 × DELIVERY_RATES[id]
기타영업비용   = 월매출 × 0.06 (전 업종 동일)

총비용        = 재료비 + 배달수수료 + 임대료(150) + 인건비(240) + 이자(44) + 기타영업비용
실수령        = 월매출 - 총비용
시급          = 실수령 ÷ 26일 ÷ 12시간
시급비율       = 시급 ÷ 10,450 × 100 (%)
폐업률        = (1 - survival_rate_3yr) × 100 (%)
```

### 카드뉴스에 포함하지 않는 항목

부가세, 종합소득세, 원금상환, 재료비 마크업, 4대보험 — 앱에서만 계산.
→ CTA 핵심: "정확한 계산은 앱에서 해보세요"

## 7. 새 업종 생성 절차

1. `businessTypes.ts`에서 해당 업종 데이터 확인
2. `calculator.ts`에서 `DELIVERY_RATES[id]` 확인
3. 위 공식으로 계산 (또는 섹션 6 계산 시트 참조)
4. `cp card-template-v2-magazine.html card-template-v2-{업종}.html`
5. 업종명·숫자 교체
6. 렌더링: `cd app && node render-cards-tmp.mjs`
7. **섹션 8 체크리스트 검증 수행**

## 8. 데이터 정합성 룰 (최우선)

> **⛔ 데이터 오류는 신뢰를 파괴한다.**

### 절대 금지

1. **추측으로 숫자 넣기 금지** — `businessTypes.ts` 또는 `calculator.ts`에서만 가져올 것
2. **`misc_fixed_cost_monthly` 사용 금지** — 반드시 `매출 × getMiscCostRate(category)` 사용
3. **공식 기관 데이터처럼 표기 금지** — "업종 평균 추정치 (AI)" 필수
4. **이전 카드뉴스 숫자 복사 금지** — 반드시 원본에서 새로 계산

### 필수 검증 체크리스트 (완성 전 반드시 수행)

- [ ] **1차**: 객단가·일손님·원가율 → `businessTypes.ts` 일치?
- [ ] **2차**: 배달수수료율 → `DELIVERY_RATES[id]` 일치?
- [ ] **3차**: 기타비용 → `getMiscCostRate(category) × 월매출`로 계산? (고정값 사용 안 했는지?)
- [ ] **4차**: 산술 → 총비용 = 재료+배달+임대+인건비+이자+기타, 실수령 = 월매출-총비용
- [ ] **5차**: 시급 = 실수령÷26÷12, 시급비율 = 시급÷10,450
- [ ] **6차**: 폐업률 = (1 - survival_rate_3yr) × 100
- [ ] **7차**: 최저시급 연도 → 현재 연도 일치?

### 주의

- 인건비 240만 = 급여만 (4대보험 미포함)
- 세금 미포함 → 실수령은 세전 영업이익
- 폐업률 출처: 국세청 국세통계 2025.04
- 2026년 최저시급: 10,450원 — 연도 변경 시 업데이트
