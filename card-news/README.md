# Card News — 사장 될 결심

카드뉴스 제작용 폴더입니다.

## 구조

```
card-news/
├── templates/       # 카드뉴스 템플릿 HTML
├── output/          # 생성된 카드뉴스 이미지
├── storylines/      # 스토리라인 기획 문서
└── README.md
```

## 사용법

1. `storylines/` 에 스토리라인 기획
2. 스크린샷(`app/screenshots/`)과 시바 캐릭터를 활용하여 카드뉴스 디자인
3. 생성된 이미지는 `output/`에 저장

## 스토리라인 예시

- **자영업 현실 시리즈**: "치킨집 사장님의 하루" — 매출, 비용, 순이익 구조를 카드로 설명
- **비교 시리즈**: "직장인 vs 자영업자" — 연봉 대비 자영업 수익 비교
- **시뮬레이션 소개**: "내 가게, 얼마 벌까?" — 앱 사용법 카드뉴스
