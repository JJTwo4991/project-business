## 선정 기준 요약

- **1차 기준**: 가맹점 수 (공정거래위원회 2023년 정보공개서 기준)
- **2차 기준**: 업종 대표성 (해당 업종에서 가장 잘 알려진 브랜드)
- **제외 기준**: 직영점 위주 브랜드 (스타벅스 등 정보공개서 미등록)
- **데이터 출처**:
  - 공정거래위원회 가맹사업 현황 통계 (2024.04 발표, 2023년 말 기준)
  - KOSIS 프랜차이즈 통계 (kosis.kr)
  - 각 업종별 전문 매체 교차 검증

## PDF 다운로드 방법

1. https://franchise.ftc.go.kr 접속
2. 좌측 메뉴 "정보공개서" → "통합검색" 또는 "브랜드별 검색"
3. 브랜드명 입력 → 검색
4. 해당 브랜드의 "정보공개서" 상세보기 → PDF 다운로드
5. 파일명: `브랜드명.pdf` 로 저장 (예: `교촌치킨.pdf`)
6. 저장 위치: `c:\Users\wows2\Downloads\franchise\` 폴더에 업종별로 정리

## 파싱 명령어

```bash
# 단일 파일
node scripts/parse-ftc-pdf.mjs "c:\Users\wows2\Downloads\교촌치킨.pdf" 치킨

# 폴더 일괄 처리
node scripts/parse-ftc-pdf.mjs --dir "c:\Users\wows2\Downloads\franchise" 치킨

# Supabase 업로드
node scripts/parse-ftc-pdf.mjs --upload
```
