### 1-2. 규모별 월세 환산 가이드 문구

앱 데이터(`rentGuide.ts`)의 ㎡당 단가에 면적을 곱해 아래 문구 형식으로 표시:

```
"[지역명] [XX]평 상가 기준, 보통 월 [YYY~ZZZ]만원 수준입니다."
예: "서울 마포구 15평(50㎡) 상가 → 보통 월 150~250만원"
```

계산 공식:
- 소규모 상가 기준 ㎡당 단가 × 면적(㎡) = 추정 월세
- 범위: 기준값 ×0.7 ~ ×1.4 (하한~상한)
- 1평 = 3.305㎡

**신뢰도:** 중간 (지역별 편차 크므로 참고치로만 사용)

## 8. 수기 기입 필요 목록

공공통계 또는 신뢰할 수 있는 데이터 출처를 찾지 못한 항목. 앱 배포 전 운영팀이 직접 조사·입력 필요.

### 8-1. 일 고객수 — 데이터 부재 업종

| 업종 | 부재 이유 | 권장 조사 방법 |
|------|----------|-------------|
| 한식당 (소규모) | 공공통계에 일 단위 고객수 없음 | 소상공인 상권정보시스템(sg.sbiz.or.kr) 상권분석 활용 |
| 분식점 | 동일 | 동일 |
| 네일샵 | 통계청 서비스업조사 미포함 | 네이버 예약 데이터 또는 업계 인터뷰 |
| 피부관리실 | 동일 | 동일 |

### 8-2. 학원 업종 지표 재정의 필요

현재 앱의 "일 고객수" 슬라이더는 학원 업종에 적합하지 않음.
학원은 "일 고객수"보다 "월 수강생 수" + "1인 수강료"로 매출 계산이 더 정확:

| 항목 | 제안값 (소규모 개인학원 기준) | 출처 |
|------|---------------------------|------|
| 월 수강생 수 | 20~80명 | 학원 창업 사례 기반 |
| 수학/영어 월 수강료 | 20~50만원 | 과목·지역 따라 편차 |
| 예체능 월 수강료 | 8~20만원 | |

### 8-3. 공인중개사 업종 지표 재정의 필요

부동산중개업은 "일 고객수" 대신 "월 계약 건수" + "건당 중개수수료"로 수익 계산이 적합:

| 항목 | 제안값 | 출처 |
|------|--------|------|
| 초기 1~2년차 월 계약 건수 | 0~5건 | eduwill 공인중개사 Q&A |
| 경력 3~5년 월 계약 건수 | 5~15건 | |
| 원투룸 전월세 중개수수료 (매물당) | 20~50만원 | 법정 요율 기준 |
| 아파트 매매 중개수수료 | 100~500만원+ | 매물 가격 따라 |

**출처:** https://namu.wiki/w/공인중개사 , https://kin.eduwill.net/board/qustnView?qustnIdx=104119
