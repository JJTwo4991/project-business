# 가이드라인 룰북

## 원칙

1. **모든 금액 입력 슬라이더에는 반드시 가이드라인 표시**
   - 사용자가 "이 값을 어떻게 설정해야 하지?" 고민하지 않도록
   - 정확하지 않아도 되니까 '대충' 얼마나 드는지 안내

2. **업종별 × 규모별로 구체적 수치 제공**
   - 예: "소규모 치킨전문점은 평균 일 25명이 방문해요"
   - 범위보다 구체적 숫자 1개가 더 도움됨

3. **출처를 반드시 표기**
   - 작은 회색 글씨로 출처 표시
   - 예: "소상공인시장진흥공단 2025 기준"

4. **자료 없는 항목은 수기 기입 목록으로 관리**
   - `src/data/guidelines.ts`에서 해당 step에 `null` 반환
   - 추후 데이터 확보 시 추가

## 가이드라인이 표시되는 스텝

| 스텝 ID | 화면명 | 가이드라인 내용 |
|---------|--------|--------------|
| `select-scale` | 매장 규모 선택 | 업종×규모별 초기투자 추정치 |
| `investment-breakdown` | 초기투자 항목분해 | 프랜차이즈 평균 창업비용 (FTC) |
| `set-investment` | 자본 구조 | 정책자금 금리 안내 |
| `set-loan` | 대출 조건 | 정책자금/시중은행 금리 안내 |
| `set-customers` | 일 방문객 | 업종×규모별 평균 고객수 |
| `set-ticket` | 객단가 | 업종별 평균 결제금액 |
| `set-labor` | 직원 수 | 최저임금 기준 인건비 |
| `set-rent` | 월 임대료 | 지역×면적 기반 추정 월세 |
| `select-region` | 지역 선택 | (지역 선택 UI 자체에 포함) |

## 데이터 소스

| 소스 | 신뢰도 | 활용처 |
|------|--------|--------|
| 공정거래위원회 가맹사업정보공개서 | 높음 | 프랜차이즈 창업비용 |
| 소상공인시장진흥공단 | 높음 | 정책자금, 상권정보 |
| 한국부동산원 | 높음 | ㎡당 임대료 |
| 고용노동부 | 높음 | 최저임금 |
| 한국소비자원 참가격 | 높음 | 외식비 메뉴별 가격 |
| 캐시노트/배달앱 | 중간 | 업종별 객단가, 주문건수 |

## 수기 기입 필요 항목

아래 항목은 공공통계 미확보로 앱 데이터에 추정치 사용 중:

- 한식당 일 고객수 (테이블 × 회전율 추정)
- 분식점 일 고객수 (빠른 회전 추정)
- 네일샵 일 고객수 (시술시간 기반 역산)
- 학원 수강생수 (일 고객수 개념 상이)
- 공인중개사 계약건수 (일 고객수 개념 상이)
